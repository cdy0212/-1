# 專案更新記錄

## 2026-04-09

### 遊戲機制更新

#### 修改摘要
- 一槍命中即殺：將子彈傷害提高到200，確保一擊必殺
- 回合目標限制：每回合最多同時出現5個目標
- 評分系統調整：擊中每個目標獲得50分（原為100分）
- 目標視覺增強：重新設計目標模型，使其更立體且易於識別

#### 具體變更
1. **game.html**
   - 增加 `bulletDamage` 從 25 提升到 200
   - 修改 `spawnTargets()` 函數，限制同時最多 5 個目標
   - 將擊中目標得分從 100 調整為 50 分
   - 完全重寫 `createTarget()` 函數，採用更立體的設計：
     - 加寬底座並增加陰影效果
     - 加粗立桿並加強陰影
     - 增大背板尺寸並改善環形圖案
     - 添加裝飾性邊框增加立體感
     - 改進健康條顯示

## 2026-04-02

### 檔案掃描初始記錄

#### 檔案清單

| 檔案名稱 | 行數 | 說明 |
|----------|------|------|
| `index.html` | 163 | 主頁面 - 特戰英豪 5v5 PVE 線上射擊遊戲，含 socket.io 多人連線支援 |
| `game.html` | 670 | 單人版頁面 - 完整單人射擊遊戲，含波次系統、跳躍、換彈機制 |
| `game.js` | 424 | 多人連線遊戲邏輯 - 使用 socket.io 實現多人同步、敵人管理、子彈系統 |

#### 技術堆疊

- **Three.js r128** - 3D 渲染引擎
- **Socket.io** - 即時多人連線通訊
- **原生 JavaScript** - 遊戲邏輯

#### 功能摘要

**index.html**
- 5v5 PVE 線上模式
- 血量條、敵人計數、玩家計數 UI
- 準心、擊殺訊息顯示
- 開始畫面與控制說明

**game.html**
- 單人射擊模式
- 波次系統 (Wave System)
- 武器系統 (VANDAL，25 發彈匣)
- 跳躍與重力物理
- 換彈機制 (R 鍵)
- 敵人 AI (移動、射擊)
- 分數系統
- Game Over 畫面

**game.js**
- Socket.io 多人連線同步
- 玩家移動、射擊同步
- 敵人生成與狀態同步
- 子彈物理與碰撞檢測
- 即時 UI 更新
